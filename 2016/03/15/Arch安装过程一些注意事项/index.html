Win10+Arch安装过程一些注意事项

Start

最近也是借着整理自己电脑的机会，重新将原来的win8.1+ubuntu双系统，成功地切换成了win10+arch。

更换的原因其实就是两年多的时间里，各种东西也是随意地丢在了电脑里面，明显感觉到硬盘不堪使用了，需要好好整理一些资料了。
所以就借了室友的移动硬盘把东西的保存起来。就开始装系统大业了。

Win10

win10其实没有难点，直接在msdn上面下载原始镜像，使用刻盘工具就可以安装了，不再详述。

Arch

提到这个东西，说多了都是泪。我不知道你要装几次，反正我是装了好几次。

首先作为一个初次接触这个Arch的人来说，一个简单的安装教程是很重要的。我比较推荐使用Enjoy arch
这篇文章。这篇文章的优点就是将wiki里面的东西整理的很好。wiki很好也很全，但是可能就是因为这一点就导致一个面向一个只想知道
安装过程的人需要读大量的wiki，自己去消化理解才能初步地去安装，这样就太累了。所以这个Enjoy arch
就比较好。

我就不想重复重复造轮子，也造不出更好的。但是可能因为时间的原因，有些事情发生了，总之这篇教程有些东西需要注意一下，才能
很好的安装。

  1. Enjoy arch —— 安装arch中配置网络中不能开启dhcpcd服务，不然会导致无线网络连接不上。
    当然如果你是用的自动分配的有线网络的话，应该没有问题。

  2. Enjoy arch —— 安装arch中配置镜像

    • Server = http://mirrors.hustunique.com/archlinux/$repo/os/$arch
    • Server = http://mirrors.tuna.tsinghua.edu.cn/archlinux/$repo/os/$arch

    先使用清华的源，后来清华的源不行了，就换另一个源。一般速度很保持上M的速度。

  3. Enjoy arch —— 基础配置中添加archlinuxcn源中

    • [archlinuxcn]
    • SigLevel = Optional TrustAll
    • Server = http://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

    必须一模一样地写入pacman.conf中。然后强制刷新archlinuxcn仓库，不然会导致yaourt无法安装。
    或者出现使用yaourt安装一些东西其实缓慢的情况。

  4. Enjoy arch —— 基础配置中安装图形界面安装xorg时，会出现4选一的情况，这个时候必须选择默认
    的情况，其实这个是wiki中OpenGL中的，必须对号入座
    。但是有一点是这样的，如果你是笔记本，那么进入Arch桌面的时候，必须是要有核显的才能进入。一般核显都是Intel，所以都是
    使用默认的mesa-libgl。之后，安装显卡驱动xf86-video-intel，这个是对应的Intel驱动。一般教程上
    都是xf86-video-nouveau，但是这个是兼容所有的驱动，所以驱动做的不是很好。

  5. 另外一般笔记本都是双显卡驱动的，这个时候必须看wiki的ble。如果，这个配置不好的话，会出现进入不了系统的情况。
    一般就是输入完密码后，直接黑屏，卡死。所以，我推荐不安装双显卡，其实核显可以的话，对于Arch来说是够了的。

The end

这些都是自己理解的东西，肯定会有不对的地方，还望见谅并指正。
